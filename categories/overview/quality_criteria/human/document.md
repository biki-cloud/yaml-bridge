### QC-001: API 応答時間

認証・ユーザープロファイル API の平均応答時間が 100ms 以下であること。
負荷テストで 95 パーセンタイル 200ms 以下を確認する。


**優先度:** 必須

**関連資料:**
- [プロジェクト概要](categories/overview/project_summary/ai/document.yaml)

### QC-002: OAuth 2.0 準拠

認証フローが RFC 6749 に準拠していること。
トークン発行・リフレッシュ・エラーレスポンスの形式を仕様書と照合する。


**優先度:** 必須

**関連資料:**
- [要件整理](categories/design/requirements/ai/document.yaml)

### QC-003: セキュリティスキャン合格

依存ライブラリの脆弱性スキャン（例: npm audit / Dependabot）で high/critical が 0 であること。
本番デプロイ前に実施し、不合格時はリリースを保留する。


**優先度:** 必須

### QC-004: 単体テストカバレッジ

認証・ユーザー管理のコアロジックについて、行カバレッジ 80% 以上を維持する。


**優先度:** 推奨

### QC-005: 受入テスト完了

PO が定義した受入条件を満たし、受入サインオフが取得されていること。


**優先度:** 必須
