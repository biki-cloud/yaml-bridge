# doc_type: document（investigation）

## この doc_type の役割

**そのカテゴリで他 doc_type に当てはまらない情報用**の汎用ドキュメントです。

- investigation には code_understanding, domain_knowledge, related_code_research, investigation_summary, open_items, tasks があるが、どれにも当てはまらない調査メモ・サマリはここに書く。
- 調査サマリ（結論・推奨・設計へのインプット）は **investigation_summary** に書くことを推奨。それ以外の「その他」の調査メモはここに記述してよい。
