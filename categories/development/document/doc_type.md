# doc_type: document（development）

## この doc_type の役割

**そのカテゴリで他 doc_type に当てはまらない情報用**の汎用ドキュメントです。

- development には implementation_plan, implementation_detail, implementation_result, pull_request, technical_debt, dependencies, environment, runbook, incident_postmortem, open_items, tasks があるが、どれにも当てはまらない開発メモ・サマリはここに書く。
- 他 doc_type に当てはまらない補足や、一時的な開発ノートもここに記述してよい。
